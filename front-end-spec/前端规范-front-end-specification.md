# 📄 前端规范 (Front-End Specification)
**BMad Project Artifact: UX Expert (Sally) Output**

### **1. 关键用户旅程与界面 (User Journeys & Screens)**

#### **2.1. PWA-1: 离线学习模式提示 (Offline Mode Indication)**

* **界面元素:** **离线通知条 (Offline Indicator)**
    * **位置:** 位于应用主导航栏或顶部的固定位置。
    * **设计:** 采用高对比度的警示色，例如 **红色背景 + 白色文字**。
    * **文字内容:** **“当前处于离线模式”**。

#### **2.2. CUS-1: 自定义内容创建流程 (Custom Content Creation Flow)**

* **流程 A: 手动卡片创建:** 主页点击 **“+ 自定义”** → 进入**卡片创建表单** (输入日语和中文内容) → **保存卡片**。
* **流程 B: Excel/CSV 批量导入:** 在卡片管理界面中提供一个 **“批量导入”** 选项，引导用户上传文件并进行列映射。

#### **2.3. INT-1: 高效复习界面设计 (Efficient Review Interface)**

* **核心交互 (Card Swiping):**
    * 使用手势（向左/右滑动）来标记 **“已掌握 (向右)”** 或 **“需复习 (向左)”**。
    * 需配备平滑的卡片堆叠动画。
* **视觉反馈 (Visual Feedback):**
    * **正确答案:** 屏幕应**短暂闪烁**（例如：0.3秒）**绿色光效**，伴随清脆的音效。
    * **错误答案:** 屏幕应短暂显示**红色警示**光效或震动。
* **内容呈现:** 卡片默认只显示日语内容（正面），用户点击/轻触后翻转显示中文释义（背面）。
